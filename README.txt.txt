Simulación:
Al ejecutar el programa se activa la animación del apartado 3.5, simulando nieve.

Barra de herramientas:
A la hora de dibujar líneas y círculos de forma predeterminada se pintarán de color blanco, pero si posamos encima el ratón en un color de la barra de herramientas y apretamos el botón de espacio las líneas y círculos
dibujados posteriormente se pintarán del color seleccionado. Si, de la misma forma, seleccionamos la herramienta para crear una página de dibujo nueva, se borrarán todas las figuras dibujadas y pintadas, la animación volverá
a estar como en un inicio y además el color seleccionado volverá a ser el predeterminado (blanco).

Línea DDA:
Para crear una línea DDA de un tamaño determinado por el programa, se tiene que pulsar la letra L del teclado y aparecerá donde se encuentra el ratón el punto de inicio de esta línea.
Si queremos dirigir la línea y determinar su punto inicial y final, debemos mantener pulsado el botón derecho del ratón y arrastrarlo hasta el punto final de la recta.

Líneas Bresenham:
Para crear las ocho líneas de los ocho octantes correspondientes de un tamaño establecido por el programa, se tiene que pulsar la letra B del teclado y el punto central de las líneas aparecerá donde se encuentra el ratón.
Si queremos dirigir la línea del primer octante y determinar su punto inicial y final, y a partir de esta se creen las 7 líneas restantes de acuerdo con la dirección y longitud de esta, debemos mantener pulsado el botón izquierdo
del ratón y arrastrarlo hasta el punto final de la recta que pertenece al primer octante.

Círculo Bresenham:
Para crear un círculo sin rellenar, se tiene que pulsar la letra X del teclado y aparecerá la circumferencia tomando la posición del ratón como el centro de esta.
Si queremos crear un círculo rellenado, se debe pulsar la letra C del teclado y aparecerá la circumferencia rellenada tomando la posición del ratón como el centro de esta.
